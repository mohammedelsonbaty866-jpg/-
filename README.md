# Smart POS Web

نظام نقاط بيع أونلاين مع Firebase | React | Tailwind CSS

## المميزات
- نقطة بيع POS مباشرة
- إدارة العملاء والموردين
- المخزون والفواتير
- تقارير مبيعات لحظية
- Dark Mode / Light Mode
- مزامنة لحظية مع Firestore

## التشغيل محليًا
1. `npm install`
2. ضع إعدادات Firebase في `.env`
3. `npm run dev`

## Build للإنتاج
`npm run build`
